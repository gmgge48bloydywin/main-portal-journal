# Main Portal Journal

## 项目简介

**main-portal-journal** 是一个用于归档与发布独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为多个独立网页的集中存放与版本管理空间。  
每个 HTML 页面均为自包含的静态文档，适合用于长期存档、快照保存或内容展示。

## 目录说明

```
.
├── pages/            # 存放所有归档的 HTML 页面
├── assets/           # 页面引用的静态资源（CSS、JS、图片等）
├── index.html        # 可选的仓库入口页面（页面索引）
└── README.md         # 本文件
```

- `pages/` 目录：每个独立 HTML 页面以子目录或直接文件形式存放，命名遵循日期或编号规则，便于检索。
- `assets/` 目录：存放页面所需的共用或独立资源，保持页面自包含性。
- `index.html`：可选，用于生成一个简单的页面列表或导航，方便浏览。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务或数据库。
- 归档内容可包括但不限于：信息页、工具页、存档快照、临时发布页等。
- 页面命名建议采用 `YYYY-MM-DD-描述.html` 格式，便于时间线管理。
- 每个页面应尽量保持资源本地化，减少外部链接依赖，以提升存档稳定性。

## 维护说明

- 欢迎提交 Pull Request 添加新的归档页面或修正已有内容。
- 提交新页面时，请确保：
  - 页面为独立 HTML 文件，内部引用资源相对路径正确。
  - 文件放置于 `pages/` 目录，并更新 `index.html`（如存在）。
- 本仓库不承诺长期在线托管，建议使用者自行备份所需页面。
- 如有版权或内容问题，请通过仓库 Issues 联系维护者。
